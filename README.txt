Instructions to use Docker:
Run the following commands in ur cmd
Create your image --> docker build -t pingouin . (here pingouin is the name of the image)
Create a container for ur image --> docker run -d -p 80:80 pingouin (u can choose a specific port if u want)
###############################################################
Run the following command to ensure the server is running --> curl localhost:80
You can also view it in the browser now by going to --> localhost:80
